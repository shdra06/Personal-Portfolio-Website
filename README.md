Personal Portfolio Website — Shivendra Prasad

This is a clean and responsive personal portfolio website built using HTML and CSS. It showcases my skills, education, projects, and contact information in a modern and visually appealing layout. The website is created as part of my Mini Project requirement.

Modern UI with light theme and soft gradients

Subtle hover effects and smooth card animations

Fixed navigation bar for easy access

Fully responsive on desktop and mobile

Profile picture card with hover transition

Clean and simple structure using only HTML and CSS
